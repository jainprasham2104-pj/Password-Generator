# Password Generator

## Description
A Python-based password generator that creates strong and random passwords to improve account security.

## Features
- Generates random passwords
- Custom password length
- Uses letters, numbers, and special characters

## Technologies Used
- Python 3
- Random Module

## How to Run
1. Download the project.
2. Open the terminal.
3. Run:

python password_generator.py
